Tirando o icone do datepicker

Choose a date MM/DD/YYYY
Primeiro passo é localizar qual a classe que contem o icon. Para isso vamos abrir a ferramenta do desenvolverdor(inspecionar) e selecionar qual parte do componente queremos acessar.

Agora que sabemos qual a classe devemos modificar, vamos modificar ela no nosso codigo. Para isso vamos criar uma classe no componente e utilizala para acessar a classe interna dele.
Agora utilizamos o ':host ::ng-deep' + o nome da classe que criamos + o nome da classe que queremos modificar para ter acesso a essa classe interna para aplicar o estilo
